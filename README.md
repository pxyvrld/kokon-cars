# 🚗 Kokon Cars - Car Rental Service

[![Live Site](https://img.shields.io/badge/Live-kokon--cars.pl-blue)](https://kokon-cars.pl)

Modern website for a car and van rental company in Przemyśl, Poland.

## 📋 About the project

Kokon Cars is a professional rental service offering:
- 8+1 passenger vans
- Passenger cars (Lexus RX 450h, Skoda Octavia)
- Commercial vehicles (Renault Kangoo)

### ✨ Features

- 📱 Fully responsive design
- 🎨 Modern user interface
- 🍪 GDPR-compliant cookie consent
- 📊 Google Analytics integration (loaded only after consent)
- 🗺️ Google Maps integration
- 💬 WhatsApp quick contact
- ♿ Accessibility (ARIA roles & labels)
- 🚀 SEO optimization

## 🛠️ Technologies

- **HTML5** - Semantic markup
- **CSS3** - Custom styles + animations
- **JavaScript (ES6+)** - Vanilla JS
- **Bootstrap 5.3.0** - CSS Framework
- **Font Awesome 6.0.0** - Icons
- **Google Fonts (Poppins)** - Typography

## 📂 Project structure

```
kokon-cars/
```

## 📱 Responsiveness

The site is fully responsive and works on:
- 📱 Mobile devices (320px+)
- 📱 Tablets (768px+)
- 💻 Desktops (1024px+)
- 🖥️ Large screens (1920px+)

## 🔍 SEO

The project includes:
- ✅ Meta tags (description, keywords, canonical)
- ✅ Open Graph protocol + Twitter Cards
- ✅ Schema.org markup (AutoRental, with geo & pricing data)
- ✅ Sitemap.xml & robots.txt
- ✅ Semantic HTML5 with proper heading hierarchy
- ✅ Alt tags and width/height for images
- ✅ Lazy loading
